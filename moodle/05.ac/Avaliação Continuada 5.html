Questão 1 (Cormen 13.1-5): Prove que o comprimento do caminho mais longo de um nó \(x\) em uma árvore rubro-negra até uma de suas folhas descendentes é, no máximo, duas vezes o do caminho mais curto do nó \(x\) até uma folha descendente. Obs.: O comprimento de um caminho é o número de arestas no caminho. Logo, todo caminho de tamanho \(n\) tem comprimento \(n-1\).

Vamos criar dois caminhos a partir de um nó qualquer, um maior e um menor.

menor = (a1, a2, ..., an)
maior = (b1, b2, ..., bm)

Pela propriedade 5, eles têm o mesmo número de nós pretos e, pela propriedade 4, nenhum nó vermelho tem um filho vermelho.

Pegando o menor caminho, o número de nós vermelhos é no máximo igual a ⌊(n - 1)/2⌋ e o número de nós pretos é no máximo ⌈(n + 1)/2⌉.

Como n é o menor caminho, sabemos que m ≥ ⌈(n + 1)/2⌉.

Vamos supor que n ≥ 2m. A partir disso, temos m ≥ ⌈(n + 1)/2⌉ ≥ ⌈(2m + 2)/2⌉ = m + 1, ou seja, uma contradição. Logo, o maior caminho tem que ter no máximo duas vezes o tamanho do menor.

Questão 2 (Cormen 13.1-6): Qual é o maior número possı́vel de nós internos em uma árvore rubro-negra com altura negra k? Qual é o menor número possı́vel? Prove cada uma de suas afirmações.

Para maximizar o número de nós internos, considere uma árvore binária completa onde os nós alternam entre vermelho e preto e tem altura 2k + 1. Em uma árvore completa binária de altura 2k, o número total de nós é 22k+1 - 1. O número de folhas é 22k, então o número de nós internos é 22k+1 - 1 - 22k = 22k - 1. Assim, a fórmula para o número máximo de nós internos é 22k - 1.

Para minimizar o número de nós internos, considere uma árvore binária completa onde todos os nós são pretos e tem altura 2k + 1. Em uma árvore binária completa de altura 2k, o número total de nós é 22k+1 - 1, e o número de folhas é 22k. O número de nós internos é então 22k+1 - 1 - 22k = 22k - 1. Portanto, a fórmula para o número mínimo de nós internos é 2k - 1.

Questão 4 (Cormen 13.2-2): Argumente por quê em uma árvore binária de busca com \(n\) nós, existem exatamente \(n-1\) rotações possíveis. Dica: tente pensar quantas arestas têm numa árvore?

Base: Para uma árvore com 1 nó, não há rotações possíveis. Portanto, o número de rotações é 0, o que corresponde a 1 - 1.

Hipótese de Indução: Suponha que qualquer árvore binária de busca com n nós tem exatamente n - 1 rotações válidas.

Passo de Indução: Vamos considerar uma árvore com n + 1 nós.

  • Se a raiz não tem filho direito: A árvore pode realizar uma rotação para a direita. O subárvore esquerda tem n nós e, pela hipótese de indução, possui n - 1 rotações. Portanto, o total de rotações é (n - 1) + 1 = n.
  • Se a raiz não tem filho esquerdo: A árvore pode realizar uma rotação para a esquerda. O subárvore direita tem n nós e, pela hipótese de indução, possui n - 1 rotações. Portanto, o total de rotações é (n - 1) + 1 = n.
  • Se a raiz tem ambos os filhos: A árvore pode realizar uma rotação para a esquerda ou para a direita, somando 2 rotações. O subárvore esquerda tem k nós e o subárvore direita tem n - k nós. Pela hipótese de indução, o total de rotações é 2 + (k - 1) + (n - k - 1) = n.

Conclusão: Em todos os casos, a árvore com n + 1 nós tem exatamente n rotações válidas, completando a prova por indução.
